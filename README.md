# AI Practice Problems

This repository contains Python solutions for various Artificial Intelligence practice problems. These problems are part of the **Artificial Intelligence** course curriculum at the **Faculty of Computer Science and Engineering (FCSE / FINKI)**.

## 🎓 Acknowledgments & Credits

All credits for the problem statements, designs, and curriculum go to the professors and assistants at **FCSE (Faculty of Computer Science and Engineering) - Skopje**. This repository is intended for practice and educational purposes related to the AI subject.

## 📂 Repository Structure

The problems are categorized based on the search and optimization techniques used to solve them:

*   **`Uninformed/`**: Contains solutions using uninformed search strategies such as Breadth-First Search (BFS), Depth-First Search (DFS), and Uniform Cost Search (UCS).
*   **`Informed/`**: Includes problems solved using heuristic-based search methods like A* (A-star) and Greedy Best-First Search.
*   **`Constraints/`**: Focuses on Constraint Satisfaction Problems (CSP), including backtracking and consistency checking.
